Nmap Recipes For Network Reconnaissance

  • Network Discovery Scan
  • Quick Port Scan
  • Full Port Scan
  • NSE Scripts

Commands contain everything after the '$' symbol. Don't type the '$'

  1. Network Discovery Scan: $nmap -sn X.X.X.X/24 -oG discovery.txt
  2. Create IP list: $grep "for" discovery.txt | cut -d " " -f 5 > ips.txt
  3. Host Quick Port Scan: $nmap -iL ips.txt -oA quickscan
  4. Full Scan with basic scripts: $nmap -sC -sV -p- -iL ips.txt -oA fullscan
